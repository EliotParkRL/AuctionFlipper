# AuctionFlipper

Steps:
1. API call
2. Parse API output
3. a) Take an item
   b) Parse the enchants, rarity, stars, gems etc.
   c) Find auction ID and store it
5. Find price of the selected item
6. Compare the prices (threshold)
7. Return auction ID
8. Type command in chat with a forge mod
9. Sell maybe?
